# -ISPC-Procesamiento-Habla-2026
Portafolio de proyectos de la materia Técnicas de Procesamiento del Habla - TSCDIA

## Objetivos de aprendizaje

Se espera adquirir conocimientos en el procesamiento del habla abordando tanto el análisis de señales de audio como el uso de modelos de inteligencia artificial. En particular, se busca comprender el funcionamiento de técnicas y herramientas aplicadas al reconocimiento, análisis y generación de voz, combinando fundamentos teóricos con el desarrollo práctico de soluciones.

## Áreas de interés

Dentro de la Ciencia de Datos, mis áreas de interés incluyen:

- Procesamiento de Lenguaje Natural (NLP)
- Redes Neuronales y Deep Learning
- Visión por Computadora
- Ciencia de Datos
- Desarrollo de Software
